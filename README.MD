# ОПИСАНИЕ

S7NER - это алгоритм по извлечению именованных сущностей из текста, созданный на основе библиотеки spacy. Мы натренировали модель ru_date_time и использовали её в комбинации с [ru_core_news_sm](https://huggingface.co/spacy/ru_core_news_sm) и процедурным извлечением стандартизированных данных для быстрой и эффективной обработки сообщений.

**ru_date_time f1 ---- 0.95**

# МОДУЛИ

Возможно расширение набора распознаваемых сущностей - наша код база содержит инструменты для генерации данных (используя цепочки или bot-inferance):

**data_builder.py** - генерация новых данных и их разметка (с возможностью добавления шума)

**chains.py** - генерация данных через случайные цепочки

**parse.py** - обработка и парсинг данных в разные форматы, сортировка и очистка датасета

**get_datetime** - не только инструменты для стандартизации даты и времени, написанных натуральным языком, но и воркфлоу для стандартизации любых подобных данных

# Quick Start

**использование**
```
from main import S7ner

ner = S7ner
message = "здравствуйте! время вылета 17:30. меня зовут григорий герасимов и я лечу в индию. мой паспорт 4018567894 мой номер телефона 89113922345 мой имейл fghjn@gmail.com "

doc = ner(message)
print(doc)
```
**Вывод**
```
('person', 'григорий герасимов')
('location', 'индию')
('time', '17:30:00', '17:30')
('passport', '4018 567894')
('phone', '+7 911 3922 345')
('email', 'fghjn@gmail.com')
```
