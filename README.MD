# ОПИСАНИЕ

S7NER - это алгоритм по извлечению именованных сущностей из текста, созданный на основе библиотеки spacy. Мы натренировали модель ru_date_time и использовали её в комбинации с ru_core_news_sm (тут ссылка на гит) и процедурным извлечением стандартизированных данных для быстрой и эффективной обработки сообщений.

ru_date_time f1 - 0.95

# МОДУЛИ

Возможно расширение набора распознаваемых сущностей - наша код база содержит инструменты для генерации данных (используя цепочки или bot-inferance):

data_builder.py - генерация новых данных и их разметка (с возможностью добавления шума)
chains.py - генерация данных через случайные цепочки
parse.py - обработка и парсинг данных в разные форматы, сортировка и очистка датасета
get_datetime - не только инструменты для стандартизации даты и времени, написанных натуральным языком, но и воркфлоу для стандартизации любых подобных данных
