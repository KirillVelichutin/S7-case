### Тренировка

В соответствие со стандартами к тренировке spaCy начиная с версии 'spaCy v3.0', тренировка модели происходит при помощи конфигурационного файла, инициализирующего создание и запуск тренировки NER модели. Тренировка происходит при помощи ner и tok2vec пайплайнов, позволяющих модели не только тренироваться распознавать сущости по паттернам, но и частично учитывать контекст.

Процесс тренировки модели:
* Запустите create_config.py, инициализирующий создание конфигурационного файла;
* Загрузите файлы с данными для тренировки в 'data/initial_data/' и запустите trainer.py, инициирующий тренировку модели в соответствие с указанными в коде параметрами.

История обучения модели сохраняется при помощи прописанного логгера и вспомогательных функций из файла visualier.py по пути 'analysis/training_history/'. В процессе тренировки, сохраняются лучшая и последняя модели spaCy в папку 'models'. Для дальнейшего использования рекомендуется использовать именно лучшую модель, так как она будет совершать меньшее количество ошибок и выполнять свою работу с большей точностью. 

###  Тестировка

Процесс тестировки лучшей модели:
* Загрузите набор данных, с тем же набором сущностей, что и в тренировочном, но отличный по их значениям в папку 'data/initial_data/';
* Укажите путь к модели (по умолчанию: 'models/model-best') и расположению вашего набора данных в файле tester.py;
* Запустите данный файл.

В ходе тестировки будут созданы 2 матрицы ошибок, а также проведён анализ работы модели в общем, по сущностям, а также по разметке границ сущностей в тестировочном наборе данных. Анализ работы модели будет выведен в консоль, а матрицы будут сохранены по пути '/analysis/test_results/'.
